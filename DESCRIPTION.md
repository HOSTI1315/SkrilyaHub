# SkrilyaHub — Брендированные Roblox скрипты

SkrilyaHub — набор Lua-скриптов с кастомным UI на базе Kairo и FluentPlus.

## Скрипты

### Survive LAVA for Brainrots
Скрипт для игры "Survive LAVA for Brainrots".

### Omega Rarities 2
Скрипт для игры "Omega Rarities 2" (EVENT + UPDATE 1).
Файл: `[EVENT + UPDATE 1] Omega Rarities 2 🍀.lua`

### Re_Rangers X
Скрипт для игры "Re_Rangers X".
Файл: `Re_Rangers X.lua`

## Дополнительно
- `Code.txt` — промокоды

## Использование
Загрузить нужный `.lua` файл через Roblox executor. UI отображается автоматически.
